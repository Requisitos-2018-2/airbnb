UC 05 - Cadastrar conta

Descrição Este caso de uso permite o usuário realizar seu cadastro na aplicação. O cadastro pode ser feito com uma conta Google, com o Facebook ou com e-mail genérico.
Ator Usuário
Pré-requisitos Acesso à internet
Fluxo Principal 1. Usuário acessa a aplicação.
2. Usuário clica em "Cadastre-se".
3. Usuário seleciona opção de cadastro entre Facebook, Google ou e-mail.
4. Usuário insere informações necessárias.
5. Usuário pressiona cadastre-se.
Pós-condição A conta do usuário é criada, o usuário é logado automaticamente

UC 06 - Realizar login

Descrição Este caso de uso permite ao usuário fazer login no airbnb. O login pode ser feito com Facebook, Google ou e-mail.
Ator Usuário
Pré-requisitos Acesso à internet
Fluxo Principal 1. Usuário acessa o airbnb.
2. Usuário clica em "Entrar".
3. Usuário seleciona entre Entrar com Facebook, Google ou e-mail.
4. Usuário pressiona "Entrar".
Pós-condição O usuário está logado

UC 07 - Realizar Logout

Descrição Este caso de uso permite o usuário fazer logout da aplicação.
Ator Usuário
Pré-requisitos Acesso à internet, estar logado
Fluxo Principal 1. Usuário está logado na aplicação.
2. Usuário clica em "Sair".
Pós-condição Usuário não está mais logado no airbnb

UC 08 - Recuperar Senha

Descrição Este caso de uso permite que o usuário recupere uma senha perdida/esquecida.
Ator Usuário
Pré-requisitos Acesso à internet, usuário possuir uma conta e estar deslogado
Fluxo Principal 1. Usuário acessa o airbnb.
2. Usuário clica em "Esqueceu a senha?"
3. Usuário insere e-mail.
4. Usuário clica em "Enviar link de redefinição".
Pós-condição E-mail contendo um link para redefinição de senha é enviado ao usuário.
