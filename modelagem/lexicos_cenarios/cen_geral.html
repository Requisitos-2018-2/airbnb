CG01 - Login em conta

Objetivos Usuário Usuário fazer login
Contexto O Usuário entra na aplicação, porém não está logado
Atores Usuário hospede, Usuário visitante, Usuário anfitrião,
Recursos Computador ou smartphone, acesso à internet, navegador web.
Pré Condição Não estar logado no Airbnb.
Episódios Usuário entra na aplicação.
Se Usuário seleciona "Entrar com facebook" ele é direcionado à uma página para que insira email e senha do facebook.
Se Usuário seleciona "Entrar com google" ele é direcionado à uma página para que insira email e senha de sua conta google.
Se tiver sucesso, Usuário é logado.
Restrições Usuário deve ter cadastro na aplicação.
Pós condição Usuário está logado no Airbnb.


CG02 - Recuperar senha

Objetivos Recuperar senha perdida/esquecida pelo Usuário.
Contexto O Usuário deseja entrar na aplicação, porém perdeu/esqueceu sua senha.
Atores Usuário hospede, Usuário visitante, Usuário anfitrião,
Recursos Computador ou smartphone, acesso à internet, navegador web.
Pré Condição Usuário perdeu/esqueceu senha para realizar o login
Episódios Usuário entra na aplicação.
Usuário clica na opção "Esqueceu a senha?" Usuário é direcionado à página "Redefinir senha"
Restrições Usuário deve ter cadastro na aplicação.
Pós condição Usuário recebe email com intruções para realizar a redefinição de senha.


CG03 - Fazer logout

Objetivos Sair do Airbnb.
Contexto Usuário está logado no Airbnb, mas deseja sair.
Atores Usuário hospede, Usuário visitante, Usuário anfitrião,
Recursos Computador ou smartphone, acesso à internet, navegador web.
Pré Condição Usuário ter feito com sucesso realizar o login
Episódios Usuário entra na aplicação.
Usuário faz login
Usuário clica em "sair"
Restrições Usuário deve estar logado.
Pós condição Usuário não está mais logado.
